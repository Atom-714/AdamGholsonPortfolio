Games

This section showcases a collection of complete games I’ve developed, each demonstrating my skills in game design, programming, and problem solving. These projects are fully playable and reflect my ability to bring ideas from concept to finished product, covering a range of genres, mechanics, and technologies.

April, 2025

One Wheel Wonder

Developed for UCMO’s annual game jam, One Wheel Wonder is a platformer centered on the theme “The Power of One.” The player navigates Mr. Onederful’s factory on a unicycle, dodging traps and maintaining balance to reclaim their stolen bicycle. I contributed to cutscene implementation, level design, and core gameplay mechanics. Our team’s creativity and execution earned 1st place in the competition.

April, 2024

Undercover Artist

Created for UCMO’s annual game jam under the theme “Midnight Mischief,” Undercover Artist is a stealth game where players sneak through the city at night to graffiti over rival tags without getting caught. Developed entirely solo, the game earned 2nd place in the competition. I’m especially proud of its PS1-inspired art direction, eerie atmosphere, and unique spray-painting mechanic, which lets players tag walls, security cameras, and even enemies to outsmart patrols.

April, 2025

Java UDP Chat Server
BattleShip

For my Computer Networking course, my team and I developed a Java-based UDP chat server that allowed multiple clients to communicate through a central server. After completing the core chat functionality, we extended the project by implementing a fully playable Battleship game, enabling clients to challenge each other and play in real time over the network. This project strengthened my understanding of UDP communication, socket programming, and managing real-time interactions between multiple clients in a networked environment.

December, 2024

Java UDP Echo Server

For my Operating Systems course, I developed a Java-based Echo Server using UDP and socket programming. The server receives messages from clients and sends the exact messages back, demonstrating the fundamentals of network communication and data transmission. This project strengthened my understanding of low-level networking, UDP protocols, and socket handling in Java, and served as a foundation for more complex networked applications like my Battleship chat server.

December, 2024

Java Lexical Analyzer

Created a simple lexical analyzer that tokenizes inputed code into keywords, operators, identifiers, and literals. Implemented the scanning and classification logic using regular expressions and modular Java components, allowing easy extension for new token types. This project strengthened my understanding of compiler design fundamentals and language processing.
